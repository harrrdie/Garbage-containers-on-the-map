RU
Инструкция по запуску:
1. Открыть папку "server"
2. Вписать свой API ключ Gemini AI (GEMINI_API_KEY=) в файл ".env"
3. Открыть cmd в корневой папке
4. Прописать следующие строчки в консоль:
cd server
npm init -y
npm install express cors dotenv @google/generative-ai
node server.js
5. Запустить файл index.html
